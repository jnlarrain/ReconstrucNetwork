# SI-QSM
* The code is for reconstructing Quantitative Susceptibility Mapping with deep neural network, training and data generation code are available. Also a template of how to make inference for testing

# References
* SI-QSM: not link yet </br>


## Requirements
* Python 3.7

## Data generation

To generate the data please use the generate_data.py script
## Training

To train please use the train.py script

## Inference

For inference please follow the structure of prediction_template.py and follow the restrictions of the paper. The weights from the paper can be downloaded from https://drive.google.com/drive/folders/1L99jN_m1SaRindXJj2tQZaUDgr83dUcJ?usp=sharing
